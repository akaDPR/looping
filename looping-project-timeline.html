LOOPING

Project Timeline & Development Plan

GPS-Based Proximity Social Networking App

Version: 1.1 | Date: March 26, 2026

Platform: Flutter (iOS + Android) + Node.js Backend

Budget: ₹2,50,000 | Duration: 14 Weeks

📱 Project Overview

Looping is a proximity-based social networking mobile application that enables users to discover, connect, and communicate with verified people nearby using GPS location services.

Core Value Proposition

Find and connect with verified, real people in your physical vicinity — at events, campuses, co-working spaces, and public places.

What We're Building

  • 20 Core Features including authentication, nearby discovery, real-time chat, voice & video calling
  • 29 Screens with polished UI/UX matching your design system
  • GPS-based discovery using MongoDB geospatial queries (500m radius)
  • Real-time chat powered by Socket.io
  • Voice & Video calling via Zegocloud/Agora SDK
  • Privacy features including screenshot protection, field visibility controls
  • Document verification via DigiLocker (Aadhaar, PAN)
  • iOS & Android deployment to App Store and Play Store

🛠 Technical Stack

Component Technology
Mobile App Flutter (Dart), Riverpod (state management)
Backend Node.js, Express.js, Socket.io (real-time)
Database MongoDB with 2dsphere geospatial indexes
Authentication JWT tokens, Twilio (SMS OTP), Nodemailer (email)
File Storage Cloudinary or AWS S3
Push Notifications Firebase Cloud Messaging (FCM via firebase-admin SDK)
Voice/Video Calling Zegocloud or Agora SDK
Hosting VPS / Railway / Render
