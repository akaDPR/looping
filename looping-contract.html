Article 5: Scope of Work - Included Features

The following 20 features are included in this contract:

  1. User Registration (5-step flow with profile, contact, identity, purpose, education, social accounts, privacy toggles)
  2. Login (Email + Password, Phone + OTP, biometric authentication)
  3. User Profile Management (view, edit, delete account)
  4. GPS-Based Nearby Discovery (500m radius using MongoDB geospatial queries)
  5. Radar/Scan UI with live user bubbles and distance calculation
  6. Connection Request System (send, accept, reject, 48-hour expiry)
  7. Real-time 1:1 Chat (text, images, message states, typing indicator via Socket.io)
  8. Voice Calling (1:1 with mute, speaker, hold controls, CallKit/ConnectionService integration)
  9. Video Calling (1:1 with PiP, camera flip, mute controls)
  10. Push Notifications (FCM for all events: messages, calls, connection requests)
  11. Screenshot & Screen Recording Protection (FLAG_SECURE on Android, detection on iOS)
  12. Privacy Controls (field visibility toggles for profile information)
  13. DigiLocker Integration (Aadhaar, PAN verification - client must obtain API access)
  14. Social Account Linking (Google, Instagram, Facebook OAuth)
  15. Connection/Contact List (with search, online indicators)
  16. Call History Log (voice and video calls with timestamps)
  17. Block / Report User (with reason selection and admin reporting)
  18. App Settings (account, privacy, notifications configuration)
  19. Delete Account (with data purging within 30 days per DPDPA 2023)
  20. App Store & Play Store Deployment (with 5 screenshots per platform)

5.1 Screens Delivered

Total screens: 29 (including splash, login, 5-step registration, home, radar, profiles, chat, voice/video call screens, settings, and all supporting screens)

Article 10: Legal Compliance

  1. Aadhaar Act 2016: Application will NOT store raw Aadhaar numbers (Section 29 compliance)
  2. Data Encryption: All personal data encrypted at rest (MongoDB encrypted storage) and in transit (TLS 1.3)
  3. DPDPA 2023 Compliance: Consent collection, right to erasure, data minimization implemented
  4. Privacy Policy: Lawyer-drafted Privacy Policy and Terms of Service must be reviewed by client's legal counsel
  5. Grievance Officer: Client must designate Grievance Officer as required by DPDPA mandate
